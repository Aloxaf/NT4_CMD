NTOSBE

Set the following user environment variables prior to launching sizzle.cmd:

SIZ_NTROOT      Path to the NT 4.5 source root (e.g. W:\trunk.x86chk)
SIZ_NTTREE      Path to the NT 4.5 binary root (e.g. W:\bin)
SIZ_NTARCH      Target build architecture [x86, amd64, arm, mips, alpha, ppc]
SIZ_NTBLD       Target build type [fre, chk]
